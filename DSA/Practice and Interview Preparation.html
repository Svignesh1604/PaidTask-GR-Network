2.) Linked list in c++


  #include 
using namespace std;

// Node structure
struct Node {
    int data;
    Node* next;
};

// Function to insert a node at the end
void insertEnd(Node*& head, int value) {
    Node* newNode = new Node{value, nullptr};

    if (head == nullptr) {
        head = newNode;
    } else {
        Node* temp = head;
        while (temp->next != nullptr)
            temp = temp->next;
        temp->next = newNode;
    }
}

// Function to display the list
void displayList(Node* head) {
    Node* temp = head;
    cout << "Linked List: ";
    while (temp != nullptr) {
        cout << temp->data << " -> ";
        temp = temp->next;
    }
    cout << "NULL" << endl;
}

// Function to free memory
void deleteList(Node*& head) {
    Node* temp;
    while (head != nullptr) {
        temp = head;
        head = head->next;
        delete temp;
    }
}

int main() {
    Node* head = nullptr;

    // Inserting elements
    insertEnd(head, 10);
    insertEnd(head, 20);
    insertEnd(head, 30);

    // Displaying list
    displayList(head);

    // Deleting the list
    deleteList(head);

    return 0;
}

3.) Stack implementation in c :

  #include 
#include 

#define MAX 5  // Maximum size of the stack

int stack[MAX];
int top = -1;

// Function to push an element to the stack
void push(int value) {
    if (top == MAX - 1) {
        printf("Stack Overflow! Cannot push %d\n", value);
    } else {
        top++;
        stack[top] = value;
        printf("%d pushed to stack.\n", value);
    }
}

// Function to pop an element from the stack
void pop() {
    if (top == -1) {
        printf("Stack Underflow! Nothing to pop.\n");
    } else {
        printf("%d popped from stack.\n", stack[top]);
        top--;
    }
}

// Function to view the top element
void peek() {
    if (top == -1) {
        printf("Stack is empty.\n");
    } else {
        printf("Top element is: %d\n", stack[top]);
    }
}

// Function to display all stack elements
void display() {
    if (top == -1) {
        printf("Stack is empty.\n");
    } else {
        printf("Stack elements are:\n");
        for (int i = top; i >= 0; i--) {
            printf("%d\n", stack[i]);
        }
    }
}

// Main function to demonstrate stack
int main() {
    int choice, value;

    while (1) {
        printf("\nStack Menu:\n");
        printf("1. Push\n2. Pop\n3. Peek\n4. Display\n5. Exit\n");
        printf("Enter your choice: ");
        scanf("%d", &choice);

        switch (choice) {
            case 1:
                printf("Enter value to push: ");
                scanf("%d", &value);
                push(value);
                break;

            case 2:
                pop();
                break;

            case 3:
                peek();
                break;

            case 4:
                display();
                break;

            case 5:
                exit(0);

            default:
                printf("Invalid choice! Try again.\n");
        }
    }

    return 0;
}

DATA STRUCTURES INTERVIEW QUESTIONS WITH ANSWERS

Data Structures - Questions and Answers

1. What is data structure?
A data structure is a format for organizing, managing, and storing data to enable efficient access and modification.
2. List out the areas in which data structures are applied extensively?
Compilers, Operating Systems, Databases, Artificial Intelligence, Simulation, Graphics, and Numerical analysis.
3. What are the major data structures used in RDBMS, Network Data Model, and Hierarchical Data Model?
RDBMS - Array/Table, Network Model - Graph, Hierarchical Model - Tree.
4. If you are using C language to implement the heterogeneous linked list, what pointer type will you use?
A void pointer (void*) is used to store different data types.
5. Minimum number of queues needed to implement the priority queue?
Two queues are required to implement a basic priority queue.
6. What is data structure used to perform recursion?
Stack is used internally to perform recursion.
7. What are notations used in evaluation of arithmetic expressions using prefix and postfix forms?
Polish notation (prefix) and Reverse Polish notation (postfix).
8. Convert expression ((a+b)*c-(d-e)^(f+g)) to equivalent prefix and postfix notations.
Prefix: - * + a b c ^ - d e + f g
Postfix: a b + c * d e - f g + ^ -
9. What are the methods available in storing sequential files?
Contiguous storage, Linked storage, Indexed sequential storage.
10. Whether linked list is a linear or non-linear data structure?
Linked list is a linear data structure.
11. Define doubly linked list?
A doubly linked list is a type of linked list where each node contains a reference to both the next and the previous node.
12. What are the issues that hamper efficiency in sorting a file?
Disk I/O time, memory limitations, algorithm complexity, and record size can affect sorting efficiency.
13. Calculate efficiency of sequential search?
On average: (n+1)/2 comparisons; in worst case: n comparisons.
14. Is any implicit argument passed to a function when it is called?
Yes, in object-oriented languages, the implicit argument 'this' is passed to member functions.
15. Parenthesis is never required in postfix or prefix expressions? Why?
Correct, because operator precedence and associativity are inherently handled by the order of operators and operands.
16. List out few of applications of tree data structure?
Expression parsing, database indexing (B-Trees), file systems, and decision-making algorithms.
17. List out few of applications that make use of multilinked structures?
Sparse matrices, file directories, and decision trees.
18. What is the type of the algorithm used in solving 8 queens problem?
Backtracking algorithm.
19. In an AVL tree, at what condition balancing is to be done?
When the balance factor of any node becomes -2 or +2 after insertion or deletion.
20. In RDBMS, what is the efficient data structure in internal storage representation?
B+ Tree is commonly used for efficient indexing and retrieval.
21. What is the difference between array and a stack?
Array is a linear structure with random access, while stack is LIFO (Last-In-First-Out) with restricted access.
22. How to check whether a linked list is circular?
Use Floyd's Cycle Detection Algorithm (slow and fast pointer method).
23. What is a node class?
A class that defines the structure and attributes of a node in linked lists or trees.
24. When can you tell that a memory leak will occur?
When memory is allocated but not deallocated after its use, resulting in loss of reference.
25. What are types of collision resolution techniques and methods used in each of the type?
Linear probing, Quadratic probing, Separate chaining, and Double hashing.
26. Which is the simplest file structure? (sequential, index, random)
Sequential file structure is the simplest.
27. What are the notations used in evaluation of arithmetic expressions using prefix and postfix forms?
Prefix notation (Polish) and Postfix notation (Reverse Polish).
28. List out few of applications of tree data structure?
Binary search trees, heaps, syntax trees, B-trees in databases, and directory structure in OS.
29. Difference between calloc and malloc?
calloc() initializes memory to zero, while malloc() leaves memory uninitialized.
30. Which file contains the definition of member function?
Member function definitions are usually written in the .cpp (source) file.
31. How is the front of the queue calculated?
By maintaining a 'front' index that points to the first element in the queue array or node.
32. Why is the isEmpty() member method called?
To check whether the stack, queue, or list contains any elements.
33. Which process places data at the back of a queue?
The enqueue() operation.
34. What is queue?
Queue is a linear data structure that follows the FIFO (First-In-First-Out) principle.
35. What does isEmpty() member method determine?
It determines if the container has zero elements.
36. What method removes values from the top of a stack?
The pop() method.
37. What method is used to place a value onto the top of stack?
The push() method.
38. How do you assign an address to an element of a pointer array?
Using the address-of operator (&), e.g., ptr[i] = &value.
39. How many parts are there in a declaration statement?
Usually two parts: data type and variable name (optionally, initialization).
40. List some of the static data structures in C?
Arrays, Structures, and Static variables.
41. Define dynamic data structure?
A data structure that allocates memory at runtime.
42. List some of dynamic data structures in C?
Linked List, Stack, Queue, Trees using dynamic memory allocation.
43. Define linear data structure?
A structure where elements are arranged in a sequential manner (e.g., Arrays, Linked Lists).
44. Define non-linear data structures?
A structure where elements are not arranged sequentially (e.g., Trees, Graphs).
45. State the different types of linked lists?
Singly linked list, Doubly linked list, Circular linked list, Circular doubly linked list.
46. List the basic operations carried out in a linked list?
Insertion, Deletion, Traversal, Searching, and Updating.
47. Define stack?
Stack is a linear data structure that follows LIFO (Last-In-First-Out) principle.
48. List out the basic operations that can be performed on a stack.
Push, Pop, Peek (Top), and isEmpty.
49. State the different ways of representing expressions.
Infix, Prefix (Polish), and Postfix (Reverse Polish).
50. What is sequential search?
Sequential search checks each element in a list one by one until the target is found or end is reached.
